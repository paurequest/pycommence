Get records from the cursor by field name and value.

Parameters:

  • field_name (str) – Name of the field to query.

  • value (str) – Value to filter by.

  • max_rtn – Maximum number of records to return. If more than this, raise CmcError.

  • empty (Literal['ignore', 'raise']) – Action to take if no records are found. Options are ‘ignore’, ‘raise’.

Return type:

list[dict[str, str]]

Returns:

A list of dictionaries of field names and values for the record.

Raises:

CmcError –

  • If the record is not found without slecting empty=’ignore’, - If max return is not None and more than max_rtn records are found.
